# Facebook phishing Awareness Demo

## Overview
This project demonstrates how phishing websites attempt to trick users into entering their credentials.  
It is **for educational purposes only** and does not connect to real services.

## Features
- Simulated login page (e.g., fake Facebook-style form).
- Admin dashboard to visualize captured test data.
- Default credentials: `admin / admin123` (for demo access only).

## Usage
1. Launch the demo server locally(ex:xampp server).
2. Navigate to the simulated login page.
3. Enter test credentials to see how data could be captured.
4. Open the admin dashboard to view submitted entries.

## Disclaimer
⚠️ This project is intended **only for training and awareness**.  
Do not deploy it publicly or use it for malicious purposes.
